# GPU Cost Simulation Idle Resources

This example simulates GPU task execution and cost calculation in a cloud environment. It demonstrates how forgetting to stop a cloud GPU instance after its work is done can lead to significant "hidden" idle charges, a common cause of unexpected high bills. The script compares a properly managed task, which is stopped promptly, with one left running idly, accumulating extra costs.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
